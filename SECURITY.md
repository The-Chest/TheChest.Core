# Security Policy

## Supported Versions
The following table indicates which versions of this library are currently supported with security updates.


| Version   | Supported          |
| --------- | ------------------ |
| 0.14.x    | :white_check_mark: |
| < 0.14.0  | :x:                |

> The next major release will be **v1.0.0**,and it might change to other security rules.

---

## Reporting a Vulnerability

If you discover a security vulnerability, **please report it responsibly** using **GitHub Security Advisories**.
